blink(1) WebHID demo simplest

This demo is for the blink(1) USB notification light

For more details on WebHID, see the WebHID awesome list

For more demos and see quickly how to set this up, see blink1-webhid

Click 'Connect' to connect to blink(1) and turn it purple.
Click 'Disconnect' to turn blin(1) off and close the device
